# 네비게이션 (Navigation)

이 디렉토리는 앱의 네비게이션 구조를 관리합니다. React Navigation 라이브러리를 사용하여 인증 상태에 따라 다른 화면으로 이동합니다.

## 구조

### RootNavigator.tsx
- 앱의 메인 네비게이션 구조
- 인증 상태에 따라 AuthStack 또는 MainTabNavigator로 분기

### AuthStack.tsx
- 로그인 이전의 화면들 (온보딩, 로그인)
- 비인증 사용자용 네비게이션 스택

### MainTabNavigator.tsx
- 인증 후의 메인 탭 네비게이션
- 모니터링 및 시스템 탭으로 구성

## 사용법

네비게이션은 사용자의 인증 상태(useAuthStore)에 따라 자동으로 라우팅됩니다. 새로운 화면을 추가할 경우, 적절한 네비게이션 스택에 등록해야 합니다.

## 특징

- SafeArea 최적화: 기기별 SafeArea에 맞춰 네비게이션 높이 자동 조정
- 인증 기반 라우팅: 토큰 상태에 따라 자동으로 화면 전환
- 탭 기반 구조: 주요 기능을 탭으로 쉽게 접근 가능
