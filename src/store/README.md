# 상태 관리 (Store)

이 디렉토리는 앱 전역 상태를 관리하는 Zustand 스토어들을 포함합니다. 인증 정보, 장비 데이터 등 앱 전체에서 공유되는 상태를 관리합니다.

## 구조

### useAuthStore.ts
- 사용자 인증 상태 관리
- 로그인/로그아웃, 토큰 관리
- 자동 로그인 및 인증 상태 확인
- 데모 모드 상태 관리

### useDeviceStore.ts
- 장비 데이터 상태 관리
- 장비 목록, 선택된 장비 정보
- 실시간 데이터 갱신

## 사용법

Zustand 훅을 통해 앱 내 어디서든 상태에 접근하고 업데이트할 수 있습니다. 상태 변경은 액션 함수를 통해 이루어지며, 상태가 변경되면 관련 컴포넌트가 자동으로 리렌더링됩니다.

## 특징

- 중앙 집중식 상태 관리: 앱 전역의 상태를 일관되게 관리
- TypeScript 지원: 타입 안전한 상태 관리
- 미들웨어 지원: 로깅, 퍼시스턴스 등 기능 확장 가능
- 성능 최적화: 불필요한 리렌더링 방지
- 보안 강화: 인증 관련 정보는 expo-secure-store와 연동
