# 설정 (Config)

이 디렉토리는 앱 전역에서 사용하는 설정 및 환경 변수들을 관리합니다.

## 구조

### env.ts
- 앱에서 사용하는 환경 변수들 정의
- API_BASE_URL: 백엔드 API 엔드포인트 URL
- IS_DEMO_MODE: 데모 모드 여부를 결정하는 플래그
- 기타 전역 설정 값들

## 사용법

환경 변수들은 앱 실행 시 `.env` 파일에서 불러와 사용됩니다. 실제 IP 주소와 같은 중요한 정보는 `.env` 파일을 통해 관리해야 합니다.

## 중요

API_BASE_URL은 실제 백엔드 서버의 IP 주소로 설정해야 하며, 이는 Docker 네트워크 설정에 따라 달라질 수 있습니다.
