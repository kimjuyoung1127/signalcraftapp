# 서비스 (Services)

이 디렉토리는 백엔드 API와의 통신, 데이터 처리 및 비즈니스 로직을 담당하는 서비스 계층을 포함합니다.

## 구조

### api.ts
- Axios 기반 HTTP 클라이언트
- 인증 토큰 자동 포함 및 401 응답 처리
- 요청/응답 인터셉터 구현

### auth.ts
- 사용자 인증 관련 서비스
- 로그인, 회원가입, 토큰 관리 기능
- expo-secure-store를 통한 안전한 토큰 저장
- 토큰 만료 및 갱신 로직

### device.ts
- 장비 데이터 관련 서비스
- 장비 목록 조회, 단일 장비 정보 가져오기
- 실제 서버 데이터와 데모 데이터 자동 전환

## 사용법

각 서비스는 싱글톤 패턴으로 구현되어 있으며, 필요한 화면에서 import하여 사용합니다. 인증이 필요한 API 호출은 자동으로 JWT 토큰을 포함합니다.

## 특징

- 보안 강화: 토큰은 expo-secure-store에 암호화하여 저장
- 자동 인증: API 호출 시 토큰 자동 포함
- 에러 처리: 네트워크 오류 및 인증 오류 처리
- 데모 모드: 실제 서버 없이도 개발 및 테스트 가능
