# 컴포넌트 (Components)

이 디렉토리는 앱에서 사용하는 재사용 가능한 UI 컴포넌트들을 포함하고 있습니다.

## 구조

### ui/
- 기본 UI 컴포넌트들 (Input, Button, StatusPill 등)
- 스타일 및 테마 관련 설정을 포함

### 기타 컴포넌트
- ScreenLayout: 앱 전체에서 사용하는 화면 레이아웃 컴포넌트
- 기타 커스텀 UI 요소들

## 사용법

모든 컴포넌트는 NativeWind CSS를 기반으로 스타일이 지정되어 있으며, Tailwind CSS 클래스를 사용하여 쉽게 스타일을 커스터마이징할 수 있습니다.
