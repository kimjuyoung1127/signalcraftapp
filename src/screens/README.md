# 화면 (Screens)

이 디렉토리는 앱의 실제 화면들을 포함하고 있습니다. 각 화면은 특정 기능 또는 사용자 작업 흐름을 담당합니다.

## 구조

### OnboardingScreen.tsx
- 앱 최초 실행 시 사용자에게 서비스 소개
- 3단계 슬라이드로 구성

### LoginScreen.tsx
- 사용자 인증을 위한 로그인 화면
- 실제 로그인과 데모 모드 전환 가능

### DashboardScreen.tsx
- 주요 모니터링 대시보드
- 장비 상태 및 오디오 분석 데이터 표시

### DeviceDetailScreen.tsx
- 개별 장비의 상세 정보 및 분석 결과
- 실시간 데이터 시각화

### SettingsScreen.tsx
- 앱 설정 및 사용자 정보 관리
- 로그아웃 기능 포함

## 사용법

모든 화면은 네비게이션 시스템과 통합되어 있으며, 상태 관리(store)와 서비스 계층(service)과 연동됩니다. 화면은 Industrial Cyberpunk 테마에 맞춰 디자인되어 있습니다.

## 특징

- 반응형 레이아웃: 다양한 화면 크기에 최적화
- SafeArea 지원: 기기별 여백 자동 조정
- 테마 일관성: 전체적으로 Dark 테마 및 네온 컬러 유지
