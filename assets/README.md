# 에셋 리소스 (assets)

이 디렉토리는 SignalCraft Mobile 애플리케이션에서 사용하는 모든 정적 리소스 파일들을 포함합니다. 아이콘, 이미지, 로고 등 앱의 시각적 요소들이 저장되어 있습니다.

## 📁 에셋 파일 목록

### 앱 아이콘 및 로고
- **icon.png** (22.4KB)
  - 메인 애플리케이션 아이콘
  - 홈 화면 및 앱 런처에서 표시되는 대표 이미지

- **logo.png** (60KB)
  - SignalCraft 브랜드 로고
  - 온보딩 화면 및 로그인 화면에서 사용
  
- **adaptive-icon.png** (17.1KB)
  - Android 적응형 아이콘
  - 다양한 Android 기기의 아이콘 호환성 보장

### 스플래시 화면
- **splash-icon.png** (17.1KB)
  - 앱 시작 시 로딩 화면
  - 초기 로딩 시 브랜드 가시성 확보

### 파비콘 (웹 버전)
- **favicon.png** (1.5KB)
  - 웹 버전에서 사용하는 작은 아이콘
  - 브라우저 탭 및 북마크에 표시

## 🎨 디자인 통합 가이드

### Industrial Cyberpunk 테마
- **컬러 팔레트**: 다크 테마 기반의 네온 컬러링
- **시각적 스타일**: 산업 현장 느낌의 미래적 디자인
- **브랜드 아이덴티티**: SignalCraft의 기술적 이미지 강조

### 이미지 포맷 표준
- **PNG 투명성**: 배경과 자연스럽게 어우러짐
- **고해상도**: Retina 및 고밀도 디스플레이 지원
- **최적화된 파일 크기**: 성능 고려한 리소스 관리

## 📱 플랫폼별 활용

### React Native 활용
```typescript
// 이미지 임포트 예시
import logo from '../assets/logo.png';

<Image source={logo} style={styles.logoImage} />
```

### 다양한 화면에서의 사용
- **OnboardingScreen**: 앱 기능 소개 시 브랜드 로고
- **LoginScreen**: 인증 화면의 브랜드 아이덴티티
- **DashboardScreen**: 장비 모니터링 대시보드
- **SplashScreen**: 앱 초기 로딩 화면

## 🔧 에셋 관리

### 이미지 최적화
- **압축**: 품질 유지하면서 용량 감소
- **형식**: PNG 포맷의 투명성 활용
- **규격**: 다양한 화면 크기 대응

### 버전 관리
- **일관성 유지**: 전체 앱에서 동일한 브랜드 이미지
- **업데이트**: 브랜드 변경 시 신속 교체
- **백업**: 원본 파일 보관 및 버전 기록

## 🚀 배포 설정

### Expo 설정 연동
```json
// app.json
{
  "expo": {
    "icon": "./assets/icon.png",
    "splash": {
      "image": "./assets/splash-icon.png",
      "resizeMode": "contain"
    }
  }
}
```

### 앱 스토어 등록
- **아이콘 요구사항**: 각 스토어별 사양 충족
- **투명도**: 다양한 배경과 조화
- **브랜드 일관성**: 모든 플랫폼에서 일관된 브랜딩
